# Reports

This directory contains reports generated by various commands.

## Report Types

| Command | Report Type | Naming |
|---------|-------------|--------|
| `/project-manage status` | Status report | `{YYYY-MM-DD-HH-MM}-status-report.md` |
| `/code-review` (ad-hoc) | Code review | `code-review-{scope}-{YYYY-MM-DD-HHMM}.md` |

## Purpose

Reports provide point-in-time snapshots of:
- Project status and health
- Code review findings
- Gap analysis results

Feature-specific reviews go in `.project/active/{feature}/` instead.
