# Research

This directory contains research documents created by `/research`.

## Naming Convention

Files are named: `{YYYYMMDD-HHMMSS}_{topic-kebab-case}.md`

Example: `20251231-143022_authentication-patterns.md`

## Purpose

Research documents capture deep exploration of:
- Codebase patterns and architecture
- Feasibility analysis
- Technical investigation
- Domain research

These documents inform `/spec` and `/design` decisions.
